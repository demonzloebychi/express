СПИСКИ:
- для отображения списка с галочкой - ul class="check-list"
- для отображения списка с цифрой - ol
- для отображения списка с точкой - ul
- для отображения списка с цифрой и вложенным списком с галочкой - ol class="include-list-check" > ul
- для отображения списка с цифрой и вложенным списком с точкой - ol > ul

БЛОКИ: 
- для отображения блока с обводкой - class="important"
- для отображения блока с обводкой и текстом посередине - class="important important-center"
- для отображения сравнения (2 блока рядом) - div class="compare" > div*2 > контент

ТЕКСТЫ:
- для увеличенного текста (обычно перед списком) - class="list-title"
- заголовки - h1, h2, h3

КАРТИНКИ:
- для отображения картинки (занимает максимум доступной ширины) - div class="full-image" > img
